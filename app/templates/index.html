Risk Gen API

About the API

Risk Gen API provides a flexible way to generate FAKE cybersecurity risk reports.

Designed for developers and security professionals to test in various applications. this API allows you to retrieve randomized risk assessments for various types of endpoints, including servers, network devices, and firewalls.

Usage

GET {{ request.url + url_for('api.gen_report').strip('/') }}

GET {{ request.url + url_for('api.gen_report').strip('/') }}?records=50

the param 'records' needs to be between 1 and 100
